
# Seminar paper: LlamaIndex Knowledge Base

Search various loaded files using LlamaIndex and InstructGPT.


## Installation

Installing the dependencies

```bash
  pip install -r requirements.txt
```

Running the app

```bash
  streamlit run app.py
```
